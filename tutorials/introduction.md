# Introducción a HR-Library

**HR-Library** es un ecosistema de desarrollo frontend diseñado para acelerar la creación de aplicaciones web robustas, consistentes y modernas.

## ¿Por qué usar HR-Library?

- **Centralización**: Todas las herramientas comunes (API, Modales, Alertas, Tablas, Fechas) bajo un mismo objeto global `HR`.
- **Automagic**: Configuración de componentes complejos (DataTables, Select2, Quill) mediante simples atributos HTML.
- **Validación Integrada**: Sistema de validación de formularios basado en clases CSS.
- **Estética Premium**: Integración nativa con Bootstrap 5 y temas personalizados.

## Componentes Incluidos

- **HTTP**: Cliente Axios con manejo de tokens y alertas automáticas.
- **UI**: Modales de Bootstrap 5, Alertas con SweetAlert2 y Notyf.
- **Data**: Integración profunda con DataTables, incluyendo exportación a Excel/PDF.
- **Forms**: Máscaras de entrada (Inputmask) y validación avanzada.
- **Rich Content**: Editores Quill, Firmas digitales y FullCalendar.
- **Utils**: Helpers para manipulación de DOM, Fechas, Números y Strings.
